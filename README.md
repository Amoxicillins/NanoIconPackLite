# NanoIconPack Lite

提供一个图标展示界面的简陋图标包APP模版。

![截图](art/screenshot.png)


### IDE兼容

**Android Studio**，没有问题。**[AIDE](https://play.google.com/store/apps/details?id=com.aide.ui)**？当然，也没有问题。


### 代码

核心代码全集中在一个 Activity 中：[LiteIconPack.java]()，这意味着你可以十分方便地把它融合到其他项目中使用。
